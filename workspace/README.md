# Workspace

This directory stores Deep Research Assistant runtime artifacts.

## Directories

- `sources/`: research inputs, plans, findings, extracted evidence, and analysis files.
- `reports/`: draft reports and final reports.

## Expected Files

Typical files generated during a research run:

- `sources/question.txt`
- `sources/research_plan.md`
- `sources/findings_*.md`
- `sources/analysis_*.md`
- `reports/draft_*.md`
- `reports/report_*.md`

These files are generated by the DeepAgents workflow through filesystem tools.
