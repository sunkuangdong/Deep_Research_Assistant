# 调研计划：LangGraph 和 AutoGen 的区别

## 研究目标
- 明确 LangGraph 和 AutoGen 两者的定义、功能和应用场景
- 对比两者的技术架构、核心特点、优势和劣势
- 分析两者在实际应用中的差异和适用范围
- 汇总权威来源和官方资料，确保信息准确可靠

## 研究方法
- 通过网络搜索获取官方文档、技术博客、权威媒体报道等资料
- 委派 researcher 子 Agent 分别调研 LangGraph 和 AutoGen 的详细信息
- 汇总调研结果，委派 analyst 子 Agent 进行对比分析
- 根据分析结果起草中文调研报告
- 委派 editor 子 Agent 审阅报告草稿，修订后形成最终报告

## 任务拆解
1. researcher 1：调研 LangGraph 的定义、功能、技术架构、应用场景
2. researcher 2：调研 AutoGen 的定义、功能、技术架构、应用场景
3. analyst：基于两份调研结果，进行结构化对比分析，突出区别和适用场景
4. 主 Agent：根据 findings 和 analysis 起草报告草稿
5. editor：审阅报告草稿，提出修改意见
6. 主 Agent：根据审阅意见修订并完成最终报告

## 时间安排
- 立即开始调研，预计调研阶段完成后1小时内完成分析和起草
- 审阅和定稿阶段预计1小时内完成

## 质量要求
- 优先使用官方原始页面和权威资料
- 明确标注信息来源和不确定性
- 报告语言专业、结构清晰、表达准确
