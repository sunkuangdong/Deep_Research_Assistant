# 调研计划：对比 LangGraph 与 AutoGen

## 目标
全面调研并对比 LangGraph 和 AutoGen 两个框架，涵盖以下方面：
1. 框架定位与核心功能
2. 技术架构与实现原理
3. 适用场景与优势劣势
4. 生态支持与社区活跃度
5. 发展现状与未来趋势

## 调研方法
- 通过网络搜索获取官方文档、技术博客、社区讨论、第三方评测等多来源信息
- 委派 researcher 子 Agent 分别聚焦调研 LangGraph 和 AutoGen 的详细信息
- 汇总调研结果后，委派 analyst 进行结构化对比分析
- 主 Agent 根据分析结果起草调研报告草稿
- 委派 editor 审阅草稿，主 Agent 根据反馈修订并完成最终报告

## 任务拆解
1. researcher 1：调研 LangGraph 框架的定位、功能、架构、应用案例等
2. researcher 2：调研 AutoGen 框架的定位、功能、架构、应用案例等
3. analyst：基于两者调研结果，进行对比分析，涵盖技术、应用、生态等维度
4. 主 Agent：起草报告草稿
5. editor：审阅报告草稿
6. 主 Agent：根据审阅意见修订并完成最终报告

## 时间安排
- 立即开始调研，预计调研阶段完成后1小时内完成分析和起草
- 审阅和定稿阶段预计1小时内完成

## 备注
- 优先使用官方来源，若官方信息不足，标注不确定性
- 保证报告结构清晰，内容专业，语言规范
