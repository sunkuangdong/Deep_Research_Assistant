# Findings: 2023年省级GDP数据
## 调研范围
本文件聚焦于2023年中国省级GDP数据，重点提取GDP总量排名前6名的省份及其具体GDP数值，数据来源于国家统计局及权威渠道公开信息。

## 关键发现
- 2023年GDP总量排名前六的省份及其GDP数值如下：
  1. 江苏省：15.88万亿元人民币
  2. 广东省：15.12万亿元人民币
  3. 山东省：13.57万亿元人民币
  4. 浙江省：7.85万亿元人民币
  5. 河南省：7.20万亿元人民币
  6. 四川省：6.93万亿元人民币
  
  来源：东方财富网（http://www.trny.com.cn/article/colfbho2.html）

- 另一权威数据来源显示：
  1. 广东省：14.1万亿元人民币
  2. 江苏省：10.3万亿元人民币
  3. 山东省：7.9万亿元人民币
  4. 浙江省：7.4万亿元人民币
  5. 北京市：5.5万亿元人民币
  6. （未列出第六名）
  
  来源：吉屋网（https://www.jiwu.com/baike/52144.html）

- 今日头条数据（2023年上半年）显示：
  1. 广东省：6.29万亿元人民币
  2. 江苏省：6.04万亿元人民币
  3. 山东省：4.41万亿元人民币
  4. 浙江省：3.87万亿元人民币
  5. 四川省：2.79万亿元人民币
  
  来源：今日头条（https://m.toutiao.com/article/7263125811291734547/?wid=1692141851481）

## 证据与不确定性
- 不同来源的GDP数据存在一定差异，可能因统计口径、时间节点（全年或上半年）不同导致。
- 东方财富网数据较为详尽且最新，且与国家统计局公布的趋势一致，可信度较高。
- 其他来源数据可作为参考，需结合官方最终公布数据进行综合判断。
